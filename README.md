# Kala Vastra E-Commerce

Kala Vastra is an e-commerce platform built using Python Flask, HTML, CSS, and JavaScript.

## Features

- Product Catalog
- Shopping Cart
- Order Management
- Coupon System
- Notifications
- Admin Panel

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Installation

```bash
pip install -r requirements.txt
python app.py
```
